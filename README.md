# soulCity_server
2018년 6월부터 2018-08까지 작업한 MO게임 습작입니다. 브로드캐스트와 몬스터 로직을 담당하는 게임서버(Netty)와 채팅서버(Akka)로 구성되어 있습니다. 클라이언트와 서버간 통신 규약으로 JSON을 사용하며, AES256으로 암호화됩니다. (매칭서버 또한 별도로 존재하나 코드가 유실되었습니다. 참고 부탁드립니다.)
